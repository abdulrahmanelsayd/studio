# Onyx | Engineering Portfolio

> A premium showcase of digital craftsmanship, architectural excellence, and engineering leadership.

## Overview

Onyx is a high-performance, visually immersive portfolio template designed for engineering leaders and senior developers. It leverages modern web technologies to deliver a seamless, responsive, and aesthetically superior user experience.

## Architecture & Structure

The codebase follows a scalable, modular architecture designed for maintainability and performance.

```
src/
├── components/
│   ├── layout/          # Global layout components (Navigation, Footer)
│   └── sections/        # Page specific sections (Hero, Projects, Contact)
├── lib/                 # Utility functions and shared logic
├── pages/               # Route entry points
└── App.tsx              # Main application root
```

## Technology Stack

### Core
- **Framework**: React 18
- **Language**: TypeScript
- **Build System**: Vite

### Styling & UI
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Components**: Radix UI (Headless accessibility primitives)

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or bun

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/onyx.git
    cd onyx
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

## Development Protocols

- **Build Production**: `npm run build`
- **Lint Code**: `npm run lint`
- **Preview Build**: `npm run preview`
